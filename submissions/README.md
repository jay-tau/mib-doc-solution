# Submissions

Each participant submits one folder named after their GitHub username:

```text
submissions/
  <your-github-username>/
    predictions.jsonl   # validation-set predictions
    MEMO.md             # 1-2 page technical memo
    SUBMISSION.md       # link to your public solution repository
```

Open a pull request that adds only your own folder, and fill out the submission
[form](https://docs.google.com/forms/d/1ZLkHmTsYd9I87JL1sUyps2rPTe6ohEI_lTZ8Jjts6bw/viewform). See "How to Submit" in the top-level `README.md`.
